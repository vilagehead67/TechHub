📘 Module 1: Introduction to Python

Python is a powerful, beginner-friendly programming language known for its simplicity and readability. It is used in various domains such as web development, data science, machine learning, automation, and more.

To get started, you need to install Python from the official website. You can write Python code using text editors like VS Code, or use beginner IDEs like Thonny or IDLE.

📘 Module 2: Variables & Data Types

Variables are like containers for storing data values. In Python, you don’t need to declare the data type – Python figures it out for you.

name = "Francis"
age = 20
height = 5.9
is_student = True

Python has the following built-in data types:

  • String: for text – "hello"
  • Integer: whole numbers – 23
  • Float: decimal numbers – 3.14
  • Boolean: True or False
